## Tools

- 機器エミュレータ
  - エアコンや照明やコントローラなどのエミュレーターです。
  - 動作環境：Windows, MacOS, Linux
  - [GitHubへのリンク](https://github.com/KAIT-HEMS/elemu)
- SSNG (Node.js version)
  - ECHONET Lite の任意のパケットを作成して送信するツールです。
  - ECHONET Lite 機器の動作確認や、ECHONET Lite プロトコルの学習に利用できます。
  - 動作環境：Windows, MacOS, Linux
  - [GitHubへのリンク](https://github.com/KAIT-HEMS/ssng-node2)
- SSNG (iOS)
  - ECHONET Lite の任意のパケットを作成して送信するツールです。
  - ECHONET Lite 機器の動作確認や、ECHONET Lite プロトコルの学習に利用できます。
  - 動作環境：Windows, MacOS, Linux
  - [App Storeへのリンク](https://apps.apple.com/us/app/ssng/id1576279020)
- Appendix Viewer
  - MRAを利用して、ECHONET Lite の仕様書「APPENDIX ECHONET機器オブジェクト詳細規定」の内容をPC常に表示します。
  - 動作環境：Windows, MacOS, Linux
  - [GitHubへのリンク](https://github.com/KAIT-HEMS/appendixViewer)
