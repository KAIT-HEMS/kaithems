## Tools

- Device Emulator
  - An emulator such as airconditioner, lighting and controller
  - OS：Windows, MacOS, Linux
  - [Link to GitHub](https://github.com/KAIT-HEMS/elemu)
- SSNG (Node.js version)
  - A tool to send ECHONET Lite packet
  - This can be used to check ECHONET Lite devices
  - OS：Windows, MacOS, Linux
  - [Link to GitHub](https://github.com/KAIT-HEMS/ssng-node2)
- SSNG (iOS)
  - A tool to send ECHONET Lite packet
  - This can be used to check ECHONET Lite devices
  - OS：Windows, MacOS, Linux
  - [Link to App Store](https://apps.apple.com/us/app/ssng/id1576279020)
- Appendix Viewer
  - MRA(Machine Readable Appendix) Viewer
  - OS：Windows, MacOS, Linux
  - [Link to GitHub](https://github.com/KAIT-HEMS/appendixViewer)
